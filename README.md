# aiuf

## Preparing (MAC)
1. Install Node and NPM 
```shell
brew install node
```
2. Install Python
```shell
brew install python
```
3. Update PIP
```shell
python3 -m pip install --upgrade pip
```
4. Install Python libraries
```shell
pip install -r requirements.txt
```
5. Node packages are preinstalled



## Running
```shell
node index.js
```

The server will listen on http://localhost:3000.
